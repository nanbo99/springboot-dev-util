## 前言
    `springboot-dev-util`项目创建于2018年5月，正在慢慢成长中，目的不仅仅是一个开发架构，而是努力打造一套从 **前端模板** 
    - **基础框架** - **分布式架构** - **开源项目** - **持续集成** - **自动化部署** - **系统监测** - **无缝升级** 
    的全方位J2EE企业级开发解决方案。
　　
## 项目介绍
### 目前项目已经整合了：
    1. rabbit-mq
    2. mybatis
    3. swagger
### 当前拥有的功能：
    1. 发送邮件
    2. 解析excel
    3. 接口文档自动生成
### 未来陆续将加入：
    1. 前段UI框架
    2. 分布式唯一主键生成工具
    3. 数据库连接性能监控工具
    4. 持续继承
    5. RPC远程框架
